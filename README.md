## ðŸ§  How It Works
1. Movie metadata is processed (genres, keywords, overview, cast, crew).
2. Text features are vectorized using **CountVectorizer**.
3. **Cosine similarity** is computed between movies.
4. On selecting a movie, the most similar movies are recommended.
